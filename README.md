# myip

返回本机的公网 IP 以及访问外网的公网 IP，使用 http://www.ip138.com 以及 http://ib.sb/geoip 提供的服务。
查看[文档](https://crystal-china.github.io/myip/)获取更多帮助

```sh
 ╰─ $ bin/myip 
 ╰─ $ myip

ip.sb/geoip：您访问外网地址信息：
```
