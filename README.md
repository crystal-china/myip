# myip

返回本机的公网 IP 以及访问外网的 IP，使用 https://getip.pub 提供的服务。

```sh
 ╰─ $ bin/myip 
国内网站：当前 IP：123.123.123.123  来自于：中国 北京 联通
国外网站：1.2.3.4
谷歌网站：1.2.3.4

```
